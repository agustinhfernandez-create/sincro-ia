# Guía de Diseño — Sincro IA

Estilo C — amigable / accesible.

## Paleta de Colores
| Rol        | Hex       | Nombre        | Uso |
|------------|-----------|---------------|-----|
| Primario   | #2D9CDB   | Azul cielo    | Headers, marca, links |
| Secundario | #1E293B   | Azul pizarra  | Nav oscuro, texto fuerte |
| Acento     | #FF8A3D   | Naranja       | Botones, CTAs |
| Fondo      | #F5F7FA   | Gris claro    | Background |
| Texto      | #1E293B   | Slate         | Texto principal |

## Tipografía
- Títulos : **Poppins** — Google Fonts (geométrica, amigable)
- Cuerpo  : **Inter** — Google Fonts (legible en UI)

## Estilo UI
- Estilo general : flat / minimalista
- Bordes          : redondeados 8px
- Iconografía     : Lucide
- Espaciado base  : 8px grid

## Referencias visuales
Ninguna provista por el usuario.

## Notas de diseño
- Aplica tanto a la web de venta (responsive desktop+mobile) como al wizard del instalador (desktop).
- Tono accesible: el público incluye no-devs, evitar jerga técnica en la UI de instalación.
- Logo: a definir más adelante.
